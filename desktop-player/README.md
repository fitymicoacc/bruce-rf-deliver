# Bruce-RF Signal Player

Упрощённое десктопное приложение для воспроизведения сигналов.
Только передача, без захвата.

## Установка

```bash
pip install bleak
```

## Запуск

```bash
python player.py
```

## Использование

1. Нажать **Connect** - найдёт и подключится к "Bruce-RF" по BLE
2. Два режима ввода:
   - **Manual Input**: выбрать протокол, ввести код (hex), биты, частоту
   - **Load from File**: загрузить JSON-файл с параметрами сигнала
3. **Play (single)** - однократная отправка (10 повторов)
4. **Hold to Send** - зажать кнопку для непрерывной передачи, отпустить для остановки

## Формат JSON-файла

```json
{
    "protocol": 1,
    "key": 10597061,
    "bits": 24,
    "pulse_length": 350,
    "frequency": 433.92
}
```

`key` - десятичное число (0xA1B2C5 = 10597061).
`pulse_length` - опционально, берётся из таблицы протоколов если не указан.
`frequency` - опционально, по умолчанию 433.92 МГц.

## Требования

- Windows 10/11, macOS или Linux с BLE
- Python 3.8+
- bleak (pip install bleak)
